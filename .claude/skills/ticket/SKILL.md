# Ticket

Compatibility wrapper for setups that expect `/ticket`.

## Mapping

- Create a new task quickly -> `/todo`
- Work with Jira directly -> `/jira` or `/jira-batch`
- Execute a ticket autonomously -> `/ticket-execute`

Choose the smallest matching workflow for the request.
