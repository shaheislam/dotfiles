# Senior Platform Engineer - Cloud Infrastructure

## About the Role
We are looking for a Senior Platform Engineer to join our Infrastructure team. You will be responsible for designing, building, and maintaining our Kubernetes-based cloud platform on AWS.

## Key Responsibilities
- Design and implement scalable Kubernetes infrastructure using EKS
- Develop Infrastructure as Code using Terraform and CloudFormation
- Build CI/CD pipelines using GitHub Actions and ArgoCD
- Implement monitoring and observability solutions using Prometheus and Grafana
- Ensure security compliance and implement best practices
- Lead migration projects from legacy infrastructure to cloud-native solutions
- Mentor junior engineers and provide technical leadership

## Required Skills
- Strong experience with Kubernetes (EKS or AKS)
- Proficiency in Infrastructure as Code (Terraform, Ansible)
- Experience with AWS services (EC2, S3, RDS, Lambda)
- Programming skills in Python or Golang
- Strong understanding of CI/CD practices and GitOps
- Experience with monitoring tools (Prometheus, Grafana, Datadog)
- Knowledge of security best practices and compliance

## Nice to Have
- AWS certifications
- Experience with cost optimization
- Experience with service mesh (Istio)
- Knowledge of FinOps practices

## What We Offer
- Competitive salary (£150K - £180K)
- Remote work options
- Professional development opportunities
